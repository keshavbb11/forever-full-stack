Forever is a full-stack e-commerce web application built using the MERN stack.  
I created this project to understand how real e-commerce platforms work, including authentication, product management, and online payments.

## Features
- User authentication using JWT  
- Role-based access (Admin & User)  
- Product CRUD operations  
- Shopping cart functionality  
- Razorpay payment integration  
- Responsive UI

- ## Tech Stack
- React.js  
- Node.js  
- Express.js  
- MongoDB  
- JWT  
- Razorpay

- ## About the Project
Users can browse products, add them to the cart, and place orders securely.  
Admin users can manage products through protected routes.
